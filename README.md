# Memory Card Game With LitElement


## Quick start

```
npm install -g polymer-cli
git clone https://github.com/GabrielUmbelino/memory-card-game-lit-element
cd memory-card-game-lit-element
npm install
polymer serve
```

### Start a dev server

```
npm install -g polymer-cli
cd memory-card-game-lit-element
npm install
```
